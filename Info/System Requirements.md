# System Requirements
## Minimum
### Operating System - Windows 10 or later. DOES NOT SUPPORT LINUX/MACOS
### Central Processing Unit - Any low-end modern CPU. Older CPUs should be Dual-Core.
### Random Access Memory - 4GB. Enough to run a web browser.
### Graphics Processing Unit - Modern CPU Integrated Graphics.
### Storage - 512MB Hard Disk or SSD. Program requires 336MB of free space.
### Network - A decent network connection.
## Recommended
### Operating System - Windows 10.
### Central Processing Unit - Quad-Core CPU.
### Random Access Memory - 8GB. Allows for multitasking with the assistant.
### Graphics Processing Unit - NVIDIA GeForce GTX 920M or Equivalent
### Storage - 512GB SSD. Program requires 336MB of free space.
### Network - LAN connection.
